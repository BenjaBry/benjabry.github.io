CYLO PERFORMANCE MOBILE V12

Objetivo:
Atacar la alerta de rendimiento móvil de PageSpeed, no solo el score.

Datos del reporte:
- First Contentful Paint: 2.9 s
- Speed Index: 4.4 s
- CLS: 0
- GTM: 175 KiB / 217 ms
- Google Fonts: 77 KiB
- Render-blocking: cylo-ui.css + Google Fonts
- DOM: 558 elementos

Cambios realizados:
1. Analytics ultra diferido:
   - Se eliminó de la ruta crítica.
   - Carga por primera interacción o después de varios segundos/idle.
2. Google Fonts deja de bloquear renderizado:
   - Ahora usa preload as=style + onload.
3. Hero móvil más liviano:
   - En móvil no carga visualmente la imagen de fondo como primera prioridad.
   - Se elimina el command panel y hero-bottom en móvil.
   - Se reduce altura y complejidad del primer pantallazo.
4. Header móvil compacto:
   - Se oculta top-bar en móvil.
   - Navbar horizontal con scroll en lugar de grid alto.
   - Menú de categorías como panel móvil.
5. Catálogo móvil:
   - Cards en 2 columnas.
   - Menor saturación visual.
   - Categorías con hash limpio sin ?cat=.
6. CSS:
   - Se corrige el min-height excesivo de #cylo-header.
   - Se incluyen versiones minificadas opcionales.
7. Vercel:
   - Headers de caché para assets e imágenes.

Instalación:
1. Reemplaza:
   - index.html
   - Catalogo-digital-2026.html
   - assets/css/cylo-ui.css
   - assets/js/cylo-ui.js
   - assets/css/cylo-blog.css
   - vercel.json

2. Opcional:
   Si quieres usar CSS minificado, cambia en tus HTML:
   assets/css/cylo-ui.css -> assets/css/cylo-ui.min.css
   assets/css/cylo-blog.css -> assets/css/cylo-blog.min.css

3. Sube a GitHub/Vercel.

4. Corre PageSpeed de nuevo en móvil.
